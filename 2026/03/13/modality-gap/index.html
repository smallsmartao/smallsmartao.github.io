Understanding the Modality Gap - An Empirical Study on the Speech-Text Alignment Mechanism of Large Speech Language Models

Understanding the Modality Gap: An Empirical Study on the Speech-Text Alignment Mechanism of Large Speech Language Models

论文信息

  • 标题: Understanding the Modality Gap: An Empirical Study on the Speech-Text Alignment Mechanism of Large Speech Language Models
  • 作者: Bajian Xiang, Shuaijiang Zhao, Tingwei Guo, Wei Zou (贝壳找房)
  • 链接: https://arxiv.org/abs/2510.12116
  • 会议: EMNLP 2025

一、研究背景

1.1 大语音语言模型（LSLMs）的崛起

端到端大语音语言模型（LSLMs）通过直接处理语音和文本输入，能够生成文本或语音输出，相比传统 pipeline 架构（ASR + LLM + TTS）具有显著优势：

  • 更低延迟
  • 更好的错误容忍性
  • 更具表现力的语音合成

1.2 核心问题：性能差距

然而，LSLMs 在语义理解任务上始终不如传统 pipeline 系统：

系统 VoiceBench 得分
Whisper-large-v3 + LLaMA-3.1-8B 79.06
LLaMA-Omni (LSLM) 37.51

尽管使用相同的底层 LLM，性能差距高达 41.55 分！

1.3 本文研究问题

本文首次系统性地研究了这一性能差距的内在机制，提出：

  • 模态差距（Modality Gap）：同一模型中，语音输入和文本输入的性能差异
  • 语音-文本对齐训练后，LSLMs 丢失了部分文本输入性能，但语音与文本之间的性能差距更加显著

二、实验设置

2.1 模型架构

模型包含三个核心组件：

  1. 语音编码器：Whisper-large-v3
  2. 语音适配器：两层线性投影，压缩 5 倍
  3. LLM 主干：LLaMA3.2-3B, LLaMA3.1-8B, Qwen2.5-1.5B, Qwen2.5-7B

2.2 训练配置

  • 数据集：637,283 样本，1,604 小时语音
  • 训练：2 epochs，AdamW 优化器，lr=2e-5
  • 微调方式：全参数微调和 LoRA (r=8, α=4.0)

2.3 性能下降分析

LSLMs 在语音输入上相对于文本输入平均下降 25%，其中：

  • 微调导致的推理能力下降：8.79%
  • 次优的语音-文本对齐：16.46%

关键发现：性能下降主要来自语音-文本模态差距！

三、粗粒度分析：序列级表征

3.1 方法

对每层表征计算：

  • Cosine Similarity：方向对齐
  • Euclidean Distance：幅度差异

3.2 关键发现

重要观察：

  1. 方向对齐（Cosine Similarity）：

    • 深层表示的 cosine similarity 持续增加
    • 说明模型在更深层实现更好的方向对齐
    • 更多训练轮次 → 更高相似度

  2. 幅度差异（Euclidean Distance）：

    • 浅层到深层，Euclidean distance 逐渐增加
    • 深层加速增长
    • 说明方向趋同但幅度分化

这一模式意味着：模型在保持模态特有表征的同时，实现了有效的多模态融合。

3.3 表征相似性与性能差距的相关性

微调方式 Cosine Similarity 与 GAP 相关性
LoRA R² = 0.75（强相关）
全参数 R² = 0.39

关键发现：

  • 更好的跨模态对齐 → 更小的性能差距
  • LoRA 约束性低秩 adaptation 保留了预训练文本表征的完整性
  • 全参数微调可能导致过拟合，弱化相关性

四、细粒度分析：Token 级对齐

4.1 单调对齐模式

在所有模型和训练范式中，token 级别相似度矩阵呈现单调路径：

  • 随着文本 token 索引增加，语音帧序列中存在一条单调对齐路径
  • 这条路径不一定严格对齐主对角线，但反映了语音和文本之间的实际时间对齐结构

统计验证：

  • 平均 Spearman 相关系数：0.85（Cosine）
  • 两种相似度度量下完美对齐的 token 比例：59%

机制分析：

  • 模型不仅在全局层面实现模态对齐
  • 还自动学习到软单调的 token 级对齐
  • 反映了模型捕捉语音时间结构到文本语义结构映射的能力

4.2 Alignment Path Score

基于 token 级对齐模式，本文提出 Alignment Path Score (APS)：

  • 量化 token 级别对齐质量
  • 与模态差距的相关性比序列级表征更强

五、干预实验

基于上述分析，本文设计了两种针对性干预策略：

5.1 Angle Projection（角度投影）

对关键 token 的语音 embedding 进行角度投影，改善方向对齐。

5.2 Length Normalization（长度归一化）

对语音 token embedding 进行长度归一化，减少幅度差异。

5.3 实验结果

在 VoiceBench 的 sd-qa 子集上，干预策略显著提升了困难样本的语音输入正确率，证明了对语音表征的针对性干预具有改进潜力。

六、总结

6.1 主要贡献

  1. 系统识别性能差距的主要来源是语音和文本输入之间的模态差异
  2. 分析内部表征，发现模态差距与表征相似度在序列级和 token 级都密切相关
  3. 首次实证证据表明针对语音表征的干预可以改善语音输入性能

6.2 核心洞察

层级 趋势 意义
序列级 Cosine ↑, Distance ↑ 方向趋同，幅度分化
Token 级 单调对齐路径 自动学习时间-语义映射
相关性 强线性相关 表征相似度可预测性能差距

6.3 实践意义

  • LoRA > 全参数微调：更适合语音-文本对齐任务
  • 干预策略有效：Angle Projection 和 Length Normalization 可改善语音输入性能
  • 表征分析有价值：内部表征相似度可作为模态差距的预测指标
